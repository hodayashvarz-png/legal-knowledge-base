# מאגר ידע משפטי

תיקייה זו מכילה את המסמכים שעליהם מתבסס הניתוח המשפטי של המערכת.

## מבנה התיקיות

```
knowledge_base/
├── פסקי_דין/          # פסקי דין מבתי משפט
├── חקיקה/             # חוקים ותקנות
├── מאמרים/            # מאמרים משפטיים
└── תקדימים/           # תקדימים חשובים
```

## פורמטים נתמכים

- PDF (.pdf)
- Word (.doc, .docx)
- טקסט (.txt)
- תמונות/סריקות (.png, .jpg, .tiff) - יעברו OCR

## כיצד להוסיף מסמכים

### אפשרות 1: העתקה ידנית
פשוט העתיקי קבצים לתיקייה המתאימה.

### אפשרות 2: הפעלת סקריפט האינדוקס
לאחר הוספת קבצים, הפעילי:
```bash
cd backend
python scripts/index_knowledge_base.py
```

## מה קורה בזמן האינדוקס?

1. המערכת סורקת את כל הקבצים בתיקיות
2. מחלצת טקסט מכל קובץ (כולל OCR לסריקות)
3. שומרת את הטקסט במסד הנתונים לחיפוש מהיר
4. יוצרת אינדקס לחיפוש סמנטי

## טיפים

- השתמשי בשמות קבצים בעברית עם מספר תיק
- ארגני בתיקיות משנה לפי נושא/ערכאה
- קבצים גדולים (מעל 50MB) עלולים להאט את האינדוקס
